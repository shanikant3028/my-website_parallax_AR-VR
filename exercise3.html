Exercise 3 Answers

1. Did you use any W3Schools templates?

I did not use any ready-made template. I designed the parallax layout myself using custom CSS properties such as background-attachment fixed and flexible layout styling.

2. Which sections did you put in your CV?

  • About Me
  • Education
  • Skills
  • Projects
  • Hobbies
  • Contact

3. Colour combinations used

I provided two themes: Default theme (black text on white background) and an alternate theme (blue text on cream background). This allows users to change the appearance dynamically.

4. Interactive elements used

  • Clickable sidebar navigation
  • Smooth scrolling
  • Theme switch buttons
  • Hover effects
  • Responsive layout

5. JavaScript explanation

JavaScript is used to switch themes. A function changes the body class when a theme button is clicked. Based on the class, CSS styles are applied instantly without page reload.

6. Background colour design

Yes. The sidebar uses a dark gradient background while the main content area uses a light translucent background. This contrast separates navigation from content and improves readability.
