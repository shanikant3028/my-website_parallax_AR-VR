Assignment Answers

1. Which section did you choose? Why?

I chose the Timetable section of my website because traditional timetables are large, crowded, and difficult to read. I wanted to redesign it so users can quickly view only relevant information.

2. Weaknesses identified

  • Too much information displayed at once
  • Difficult to find specific class
  • No highlighting of current class
  • No distinction for free periods
  • Static layout

3. Main ideas for improvement

  • Interactive filtering by day and program
  • Dynamic display of classes
  • Highlight current class
  • Highlight free time
  • Clickable subjects for details

4. Interactive elements used

  • Dropdown menus
  • Clickable table rows
  • Keyboard shortcut navigation
  • Hover effects
  • Dynamic content loading
  • Live clock

5. Two improvements after redesign

Filtered timetable: Only selected day and program are shown, reducing clutter.

Highlighted free time: Break periods are visually distinct so users can quickly identify them.

6. Mouse and keyboard events used

  • onclick
  • mouseenter
  • mouseleave
  • keydown

7. Difference between mouseover, mouseenter and mouseleave

Mouseover triggers when the cursor enters an element or its child elements. Mouseenter triggers only once when entering the element itself. Mouseleave triggers when the cursor exits the element. Mouseenter and mouseleave are preferred because they do not trigger repeatedly.
